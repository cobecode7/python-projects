
# متجر إلكتروني احترافي

مشروع متجر إلكتروني متكامل مبني باستخدام Django، مع واجهة مستخدم حديثة وميزات متقدمة.

## المميزات الرئيسية

- نظام إدارة متكامل للمنتجات مع متغيرات
- نظام طلبات متقدم مع تتبع الشحن
- نظام دفع آمن ومتكامل
- نظام تقييمات ومراجعات للمنتجات
- نظام الكوبونات والخصومات
- قائمة الرغبات والمفضلة
- نظام إشعارات متقدم
- تحسينات الأداء والتخزين المؤقت
- نظام بحث متقدم
- لوحة تحكم إحصائيات وتحليلات

## الهيكل الأساسي للمشروع

```
ecommerce/
├── apps/
│   ├── products/     # لإدارة المنتجات
│   ├── orders/       # لإدارة الطلبات
│   ├── users/        # لإدارة المستخدمين
│   ├── payments/     # لإدارة الدفعات
│   ├── reviews/      # لإدارة التقييمات
│   ├── coupons/      # لإدارة الكوبونات
│   └── notifications/ # لإدارة الإشعارات
├── config/
│   ├── settings/
│   │   ├── base.py       # الإعدادات الأساسية
│   │   ├── development.py # إعدادات التطوير
│   │   └── production.py  # إعدادات الإنتاج
├── static/           # الملفات الثابتة
├── media/           # الملفات المرفوعة
├── templates/       # قوالب HTML
└── requirements.txt  # الاعتماديات
```

## المتطلبات

- Python 3.11+
- Django 4.2+
- PostgreSQL (للإنتاج)
- Redis (للتخزين المؤقت والجلسات)
- Elasticsearch (للبحث المتقدم)

## التثبيت والتشغيل

### المتطلبات الأولية
- Python 3.11+
- PostgreSQL
- Redis
- Node.js (اختياري)

### تثبيت وتشغيل المشروع

1. **نسخ الملفات المطلوبة:**
   ```bash
   cp .env.example .env
   ```

2. **تعديل إعدادات .env حسب بيئة العمل الخاصة بك**

3. **تثبيت الاعتماديات:**
   ```bash
   pip install -r requirements.txt
   # أو باستخدام الإعداد التلقائي:
   ./setup.sh
   ```

4. **إعداد قاعدة البيانات:**
   ```bash
   python manage.py migrate
   ```

5. **إنشاء مستخدم مدير:**
   ```bash
   python manage.py createsuperuser
   ```

6. **تشغيل الخادم:**
   ```bash
   python manage.py runserver
   ```

### إعداد بيئة التطوير باستخدام Docker (اختياري)

يمكنك أيضًا استخدام Docker لتشغيل المشروع:

```bash
# تشغيل الخدمات المطلوبة (PostgreSQL، Redis)
docker-compose up -d

# في نافذة جديدة، تشغيل تطبيق Django
python manage.py runserver
```
